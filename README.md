# User Dashboard Application

##  Overview

This project is a React-based User Dashboard Application.
It is being built as part of a frontend assignment to demonstrate React fundamentals, component reusability, API integration, and clean project structure.

---

##  Current Status

Initial project setup completed using Vite with React.


## Tech Stack

* React (Functional Components)
* Vite
* JavaScript (ES6+)


##  Getting Started

### 1. Install dependencies

npm install


### 2. Run development server

npm run dev

### 3. Open in browser

http://localhost:5173
