# User Dashboard Application

##  Overview

The User Dashboard Application is a React-based web application that allows users to manage user data efficiently. It demonstrates core frontend concepts such as reusable components, API integration, routing, and state management using React Hooks.

---

## Features

# User Management

View list of users
View detailed user information
Add new users
Edit existing users
Delete users

# Search & Filter

Debounced Search users by name
Sort users alphabetically (A–Z / Z–A)

# Functional Highlights

Reusable component architecture
Form validation for user input
Loading and error handling
Responsive UI design


## Tech Stack

* Frontend: React (Functional Components)
* Build Tool: Vite
* Styling: Tailwind CSS
* Routing: React Router
* API Integration: Axios
* Backend (Mock): MockAPI
